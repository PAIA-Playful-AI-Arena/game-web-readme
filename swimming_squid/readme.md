# 基本玩法

<br />

在遊戲時間截止前，盡可能吃到愈多的食物吧！

<br />

`遊戲目標` &nbsp;&nbsp;&nbsp; 時間結束前，吃到的食物超過指定分數，即可過關。

`失敗條件` &nbsp;&nbsp;&nbsp; 時間結束前，吃到的食物少於指定分數，即算失敗。

<br />

# 魷魚

- 角色初始等級皆為 1，隨著得分增加升 / 降級。等級將會影響角色長寬與移動速度。

# 食物和垃圾

- 角色可以透過吃海裡漂浮的東西獲取分數，但海裡也有垃圾存在，吃到垃圾將會扣分。
- 食物數量會隨遊戲時間增加。
- 不同的食物或垃圾會有不同的大小與分數

# 座標系統

- 所有座標皆回傳物件 `中心` 座標。
- 所有的尺寸單位為 `px` 。
- 魷魚等級將會影響角色長寬與移動速度，各等級對應資料如下：

| Lv  | 分數      | 角色寬度  | 角色高度 |移動速度 |
| --- |------     | ------- | ------ | ------ |
| 1   | 0~29      | 30      | 60     |30      |
| 2   | 30~59     | 36      | 72     |26      |
| 3   | 60~99     | 42      | 84     |22      |
| 4   | 100~139   | 48      | 96     |17      |
| 5   | 140~179   | 54      | 108    |12      |
| 6   | 180~      | 60      | 120    |6       |

- 不同的食物或垃圾會有不同的大小與分數。資料如下：

| 食物名稱 | 物件寬度 | 對應分數 |
| -------- | -------- | -------- |
| 蝦子     | 30       | 1        |
| 藍色魚   | 40       | 2        |
| 金色魚   | 50       | 4        |
| 蘋果     | 30       | -1       |
| 罐頭     | 40       | -4       |
| 咖啡杯   | 50       | -10      |
