# 基本玩法

<br />

在遊戲時間截止前，盡可能吃到愈多的食物吧！

<br />

`遊戲目標`

- 時間結束前，吃到的食物超過指定分數，即可晉級下一關。
- 若兩人同時通關，分數較高者勝。
- 若兩人同時通關且同分，遊戲將進入延長賽：提高指定分數  50 分，並且延長遊戲時間 600 frame。
- 遊戲最多延長 3 次。

`失敗條件`

- 時間結束前，吃到的食物少於指定分數，即算失敗。
- 若兩人皆未能達成指定分數，分數較高者勝。
- 若兩人皆未能達成指定分數且同分，遊戲將進入延長賽：指定分數維持不變，並且延長遊戲時間 300 frame。
- 遊戲最多延長 3 次。

<br />

# 遊戲系統

**魷魚**

- 角色初始等級皆為 1，隨著得分增加升 / 降級。等級將會影響角色長寬與移動速度。

**食物和垃圾**

- 角色可以透過吃海裡漂浮的東西獲取分數，但海裡也有垃圾存在，吃到垃圾將會扣分。
- 食物數量會隨遊戲時間增加。
- 不同的食物或垃圾會有不同的大小與分數

# 座標系統

- 所有座標皆回傳物件 `中心` 座標。
- 所有的尺寸單位為 `px` 。
- 魷魚等級將會影響角色長寬與移動速度，各等級對應資料如下：

| Lv  | 角色寬度 | 角色高度 | 移動速度 |
| --- | -------- | -------- | -------- |
| 1   | 30       | 45       | 25       |
| 2   | 36       | 54       | 21       |
| 3   | 42       | 63       | 18       |
| 4   | 48       | 72       | 16       |
| 5   | 54       | 81       | 12       |
| 6   | 60       | 90       | 9        |

- 不同的食物或垃圾會有不同的大小與分數。資料如下：

| 食物名稱 | 物件寬度 | 對應分數 |
| -------- | -------- | -------- |
| 蝦子     | 30       | 1        |
| 黃色魚   | 40       | 2        |
| 藍色魚   | 50       | 4        |
| 湯匙     | 30       | -1       |
| 薯條     | 40       | -4       |
| 空罐頭   | 50       | -10      |
